# 魔法小课桌 - 产品需求文档

## 1. 产品概述
魔法小课桌是一款面向小学生的智能学习助手应用，以魔法为主题，通过可爱的小精灵多比引导孩子学习，提供互动式学习体验。

### 1.1 产品目标
- 激发小学生的学习兴趣，让学习变得有趣
- 提供个性化的学习辅导和练习
- 帮助学生管理学习时间和课程安排
- 培养学生的学习习惯和专注力

### 1.2 目标用户
- 小学生（6-12岁）
- 关注孩子学习的家长

## 2. 核心功能

### 2.1 智能聊天助手
- **功能描述**：用户可以与小精灵多比进行自然语言对话，获取学习帮助和辅导
- **使用场景**：解答学科问题、提供学习建议、激发学习兴趣
- **技术实现**：基于AI模型的流式聊天响应，支持文本和语音输入

### 2.2 课程表管理
- **功能描述**：用户可以查看和管理课程安排，支持周视图和日视图
- **使用场景**：了解每日课程安排、添加新课程、接收课程提醒
- **技术实现**：本地数据存储，智能提醒系统

### 2.3 互动练习系统
- **功能描述**：根据用户需求生成个性化练习题，提供即时反馈
- **使用场景**：巩固知识点、测试学习成果、提升解题能力
- **技术实现**：动态生成练习内容，实时评分和解析

### 2.4 成就系统
- **功能描述**：记录用户的学习成就，包括积分、等级、任务完成情况
- **使用场景**：激励学习积极性、展示学习成果、设置学习目标
- **技术实现**：积分系统、等级晋升机制、任务管理

### 2.5 专注工具
- **功能描述**：提供专注学习的工具，包括倒计时、背景音效
- **使用场景**：集中注意力学习、培养时间管理能力
- **技术实现**：计时器、白噪音播放、标签页切换检测

### 2.6 魔法绘图
- **功能描述**：根据用户需求生成相关图片，辅助学习理解
- **使用场景**：理解抽象概念、增强学习体验
- **技术实现**：基于文本描述生成图片

## 3. 页面设计

### 3.1 主界面
- **布局**：左侧魔法咒语库，中央聊天区域，右侧魔法展示窗
- **元素**：
  - 顶部：应用标题、用户信息、功能按钮
  - 左侧：常用功能快捷入口（课程表、作业、学单词等）
  - 中央：聊天消息列表、输入框、附件上传
  - 右侧：根据选择展示不同内容（课程表、练习、成就、专注工具）

### 3.2 登录/注册界面
- **布局**：居中卡片式设计
- **元素**：
  - 魔法主题的登录/注册表单
  - 用户名、密码输入框
  - 登录/注册按钮
  - 切换登录/注册模式的链接

### 3.3 移动端适配
- **布局**：底部导航栏，内容区域自适应
- **元素**：
  - 底部导航：对话、书库、我的
  - 顶部：应用标题、用户信息
  - 内容区域：根据选择的功能展示对应内容

## 4. 技术实现

### 4.1 前端技术栈
- React 18
- Next.js 14
- Tailwind CSS
- Lucide React（图标库）
- Motion React（动画效果）
- React Markdown（Markdown渲染）

### 4.2 后端服务
- API路由：处理聊天、用户认证、数据存储等请求
- 模拟响应：为聊天功能提供模拟数据
- 本地存储：使用浏览器存储保存用户数据

### 4.3 数据结构
- User：用户信息，包括积分、等级、任务等
- Course：课程信息
- Achievement：成就信息
- KnowledgePoint：知识图谱节点

## 5. 界面设计风格
- **主色调**：深棕色背景（#0a0502），橙色强调色（#ff4e00）
- **设计元素**：
  - 玻璃态面板（backdrop-blur）
  - 渐变背景效果
  - 流畅的动画和过渡
  - 魔法主题的图标和元素
- **字体**：
  - 无衬线字体：Inter
  - 衬线字体：Cormorant Garamond（用于标题和强调）
- **视觉风格**：神秘、温馨、趣味性，适合小学生使用
